# Sliding Sidebar Menu

A modern, animated sliding sidebar menu built with **HTML**, **CSS**, and **JavaScript**.  
This project features smooth animations, hover effects, and an interactive open/close button to enhance user experience.

---

## 🌟 Features

- **Sliding Sidebar:** Smooth slide in/out animation when opening or closing the sidebar.
- **Interactive Buttons:** Animated open (`☰ Menu`) and close (`✖`) buttons.
- **Hover Effects:** Menu items and buttons have smooth hover animations.
- **Responsive Layout:** Works well on both desktop and mobile screens.
- **Professional Design:** Rounded corners, shadows, and color transitions for a modern look.
- **Content Shift Animation:** Main content moves smoothly when the sidebar is hidden.
