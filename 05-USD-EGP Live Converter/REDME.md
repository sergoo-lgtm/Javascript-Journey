## 📌 Overview

A clean and modern **Currency Converter Web App** built using pure HTML, CSS, and JavaScript.

The application allows users to convert between **USD and EGP** instantly with real-time updates and smooth UI interactions.

---

## ✨ Features

- 🔁 Real-time currency conversion
- 🎨 Modern card-based UI design
- ✨ Smooth focus animations
- 🔢 Automatic formatting (2 decimal places)
- 🛡 Input validation
- 📱 Fully responsive layout

---

## 🧠 How It Works

- When the user types inside the **USD input**, the value is multiplied by the exchange rate.
- When typing inside the **EGP input**, the value is divided by the exchange rate.
- The exchange rate is currently set to:

```
1 USD = 47 EGP
```

You can easily modify the rate inside the `script.js` file.
