# 🎯 Guess My Number!

Simple number guessing game built with **HTML, CSS, and JavaScript**.

The player has to guess a secret number between **1 and 20**.

---

## 🚀 How To Play

1. Enter a number between 1 and 20.
2. Click **Check!**
3. The game will tell you:
   - 📉 Too Low
   - 📈 Too High
   - 🎉 Correct Number
4. Your score decreases with every wrong guess.
5. If you win, the highscore updates.
6. Click **Again!** to restart the game.

---

## 🧠 Features

- Random number generation using `Math.random()`
- Score system
- Highscore tracking
- Dynamic UI changes
- Clean retro design
