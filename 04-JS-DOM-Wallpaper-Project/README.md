# 🖼️ Cool Wallpapers Gallery


A **dynamic wallpaper gallery** fully built using **JavaScript**, where the entire page content (title, cards, images) is generated programmatically.  
Perfect for practicing **DOM manipulation**, **animations**, and **interactive UI**.

---

## 🚀 Features

- 🌌 **Modern dark gradient background**  
- 🖼️ **Dynamic wallpaper cards** generated from JS arrays  
- ✨ Smooth **fade-up animations** on load  
- 🔍 **Hover effects**: scale & shadow  
- 🔄 Fully dynamic: add more wallpapers by updating JS arrays  
- 📜 Smooth scrolling and clean layout  

---

## 🧩 How It Works

1. Arrays store all data:
```js
let names = ['wallpaper1','wallpaper2','wallpaper3',...];
let resolution = ['4k','2k','HD',...];
let images = ['i1.jpg','i2.jpg','i3.jpg',...];
```
2. JS creates **all HTML elements dynamically**:
   - `<h1>` title  
   - `<div>` container  
   - Cards with `<h2>`, `<p>`, `<img>`  
3. Cards are animated and styled via JS.  
4. Everything appended to the container in a **loop**.
