# 🪟 Modal Window Project

A simple modal window project built using **HTML, CSS, and JavaScript**.

This project demonstrates how to open and close a modal using buttons, overlay, and keyboard events.

---

## 🚀 Features

- Open modal using multiple buttons
- Close modal using:
  - ❌ Close button
  - 🖱️ Clicking outside (overlay)
  - ⌨️ Pressing Escape key
- Smooth UI design
- Overlay background blur effect

---

## 🧠 Concepts Practiced

- `querySelector` & `querySelectorAll`
- Event Listeners
- Looping through NodeList
- Adding & Removing classes
- Keyboard events (`keydown`)
- Clean UI state management

---

## 🕹️ How It Works

1. User clicks any **Show Modal** button.
2. The `.hidden` class is removed from:
   - `.modal`
   - `.overlay`
3. Modal appears centered on screen.
4. When user clicks close / overlay / presses Escape:
   - `.hidden` class is added again.
   - Modal disappears.
